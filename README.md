Simple GUI application that clears your clipboard by just focusing on the window.

## How to use?

It will clear the clipboard once you opened it, restore and minimize the window can clear the clipboard quickly. An gif will show if the clipboard is cleared successfullly, the window will stay dark if the clipboard is already cleared.

# Development

- [Environment setup](https://tauri.app/) (required)

```shell
# run the application
pnpm install
pnpm dev

# build the app
pnpm build
```
